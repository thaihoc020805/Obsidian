## **Resources (Tài nguyên)**

### Document repositories (Kho tài liệu)

- **Ví dụ**: Server MCP kết nối với Google Drive, cho phép AI truy cập và đọc các tài liệu PDF, Word, Excel được lưu trữ
- **Ứng dụng**: "Tìm giúp tôi báo cáo tài chính Q3 trong folder Company Reports"

### Knowledge bases (Cơ sở kiến thức)

- **Ví dụ**: Database chứa toàn bộ documentation của một framework như React hoặc Python
- **Ứng dụng**: "Giải thích cách sử dụng React Hooks dựa trên tài liệu chính thức"

### Structured data sources (Nguồn dữ liệu có cấu trúc)

- **Ví dụ**: Kết nối với database SQL chứa thông tin sản phẩm, khách hàng, đơn hàng
- **Ứng dụng**: "Cho tôi biết doanh thu từ khách hàng VIP trong tháng này"

### File systems (Hệ thống file)

- **Ví dụ**: Truy cập local file system để đọc/ghi file trên máy tính
- **Ứng dụng**: "Đọc file config.json trong thư mục /project/settings"

## **Prompts (Mẫu câu lệnh)**

### Pre-defined conversation templates (Mẫu hội thoại định sẵn)

- **Ví dụ**: Template để phỏng vấn ứng viên với các câu hỏi chuẩn theo vị trí
- **Ứng dụng**: "Bắt đầu phỏng vấn cho vị trí Senior Developer"

### Guided interaction patterns (Mẫu tương tác có hướng dẫn)

- **Ví dụ**: Workflow hướng dẫn debug code theo các bước: xác định lỗi → tìm nguyên nhân → đề xuất giải pháp
- **Ứng dụng**: "Giúp tôi debug lỗi này theo quy trình chuẩn"

### Specialized dialogue structures (Cấu trúc hội thoại chuyên biệt)

- **Ví dụ**: Template để viết email chuyên nghiệp với các phần: greeting, context, request, closing
- **Ứng dụng**: "Viết email xin nghỉ phép theo format công ty"

## **Tools (Công cụ)**

### Data processing utilities (Tiện ích xử lý dữ liệu)

- **Ví dụ**: Tool chuyển đổi CSV sang JSON, merge nhiều file Excel, clean data
- **Ứng dụng**: "Gộp 3 file báo cáo Excel này và loại bỏ dữ liệu trùng lặp"

### External API integrations (Tích hợp API bên ngoài)

- **Ví dụ**: Kết nối với API thời tiết, tỷ giá, stock market, hoặc Twitter API
- **Ứng dụng**: "Kiểm tra giá cổ phiếu AAPL hiện tại qua Yahoo Finance API"

### Computational capabilities (Khả năng tính toán)

- **Ví dụ**: Tool thực hiện phân tích thống kê, machine learning predictions, hoặc tính toán phức tạp
- **Ứng dụng**: "Phân tích xu hướng doanh thu và dự đoán 3 tháng tới"

### Search functionality (Chức năng tìm kiếm)

- **Ví dụ**: Tool tìm kiếm trong codebase, documentation, hoặc internal wiki
- **Ứng dụng**: "Tìm tất cả các function liên quan đến authentication trong project"
